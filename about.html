The Project

The Canadian Accountability Project is a public documentation effort. Every record on this site is drawn from:

  • Auditor General reports — the AG is an Officer of Parliament with full audit access to federal departments
  • Hansard — the official verbatim record of parliamentary proceedings
  • Commission of Inquiry reports — O'Connor, Mass Casualty Commission, etc.
  • Court documents — Supreme Court decisions, Federal Court filings
  • Official Health Canada data — annual MAID reports
  • United Nations communications — formal letters and concluding observations to Canada

No anonymous sources. No speculation. If it's on this site, a primary source document supports it — and that source is cited.

The AI Infrastructure

This site uses LIRIL — a local NPU-based classifier and router — to analyze and categorize all content. LIRIL runs on this machine and communicates via NATS message bus. There is no connection to any external AI service, no API key, no cloud processing.

LIRIL NPU

Multi-probe hash classifier running on NPU. Classifies content into 6 ARTSTEM domains. SEED: 118400. TICK: 118.4Hz. 16,998 classifications, 1,564 training samples.

NemoClaw GPU

Dual RTX 5070 Ti. GPU0: Nemotron-Nano-9B (reasoning). GPU1: Mistral-Nemo-12B (complex analysis). Both with Llama-3B speculative decoding. Accessible via NATS mercury.infer.

SATOR Grid

5×5 palindromic routing grid. All accountability content routes to row 3 (OPERA) → NemoClaw. Each row corresponds to a domain: ART, TECHNOLOGY, SCIENCE, ETHICS (OPERA), MATHEMATICS.

NATS Mesh

All agent communication via NATS at 127.0.0.1:4223. 21 active connections. LAN-accessible at 0.0.0.0:4223. Monitoring at 127.0.0.1:18222.

Editorial Principles

  1. Primary sources only. Every claim has a cited official document.
  2. No anonymous attribution. Named individuals are named because official public records name them.
  3. Opinion is labelled. Investigative analysis is marked as opinion/analysis. Factual records are documented findings.
  4. No external dependencies. No tracker, no analytics, no CDN, no API. The site is fully self-contained.
  5. LIRIL analysis is transparent. Every AI classification result is shown with its confidence score, device (NPU/CPU), and method. Nothing is hidden.

What This Site Is Not

This is not legal advice. This is not a criminal complaint. This is not a demand for any specific action. It is a documented record, assembled from public sources, organized so that Canadians can review the evidence and draw their own conclusions about what their institutions did and did not do.

“Democracy requires an informed citizenry. An informed citizenry requires accessible, documented records. This site is an attempt to build one.”
